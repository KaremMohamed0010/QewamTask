## Qewam Task

the collection on the root folder 

1- install packages 
```bash
composer install
```
2- install migrate table
```bash
php artisan migrate  
```
3- install passport

```bash
php artisan passport:install
```
4- seed the database

```bash
php artisan db:seed
```

5- serve the app 

```bash
php artisan serve
```
